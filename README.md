<h1>Pasos a seguir</h1>
<section>
  <h2>1. Clonar el repositorio</h2>
  <p>Abrí la terminal, ubicáte en una carpeta de trabajo y escribí:</p>
  <code>git clone https://github.com/FranT18/ClaseGit.git</code><br>
  <code>cd ClaseGit</code>
</section>
<section>
  <h2>2. Crear una rama con tu nombre</h2>
  <code>git checkout -b tuNombre</code>
  <p>Ejemplo: <code>git checkout -b juan</code></p>
</section>
<section>
  <h2>3. Editar archivo</h2>
  <p>Abre el archivo participantes.txt con tu editor de preferencia y agregá tu nombre al final del archivo:<br><br>
  Lista de participantes:<br>
  - Cristian<br>
  - Nahuel<br>
  - Francisco<br>
  - Juan<br><br>
  Guardar y cerrar.</p>
</section>
<section>
  <h2>4. Confirmar los cambios</h2>
  <code>git add participantes.txt</code><br>
  <code>git commit -m "Agrego mi nombre al archivo de participantes"</code>
</section>
<h2>5. Enviar la rama al repositorio remoto</h2>
<code>git push origin tuNombre</code>>
<h2>️6. Crear un Pull Request</h2>
<ol>
 <li>Entrá al repositorio en GitHub.</li>
 <li>Andá a la pestaña Pull requests.</li>
 <li>Hacé clic en New pull request.</li>
 <li>Seleccioná tu rama (tuNombre)</li>
</ol>
